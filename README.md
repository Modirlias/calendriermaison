# 📅 Planning Familial Gothique

Une application web élégante et sombre pour gérer un planning mensuel familial avec style gothique vintage.

## ✨ Caractéristiques

- **Planning Mensuel** : Visualisez chaque mois avec navigation facile entre les mois et les années
- **Multi-utilisateurs** : Créez plusieurs utilisateurs avec une couleur spécifique et un genre
- **Rendez-vous Catégorisés** : 9 catégories de RDV avec icônes et couleurs distinctes
- **Genres Complets** : Différenciez les membres humains (femme/homme) des 11 animaux de compagnie
- **Style Moderne Lila** : Interface douce avec nuances de lila et violet pâle
- **Stockage Local** : Toutes les données sont sauvegardées localement dans votre navigateur
- **Responsive** : Fonctionne sur ordinateur, tablette et mobile

## 🚀 Démarrage Rapide

1. **Ouvrez l'application** : Ouvrez simplement le fichier `index.html` dans votre navigateur
2. **Créez des utilisateurs** : Cliquez sur "➕ Ajouter un utilisateur" pour créer les membres de votre foyer
3. **Choisissez une couleur** : Chaque utilisateur peut choisir sa propre couleur
4. **Ajoutez des rendez-vous** : Cliquez sur une journée pour ajouter un rendez-vous

## 📖 Guide d'Utilisation

### Ajouter un utilisateur
- Cliquez sur le bouton "➕ Ajouter un utilisateur"
- Entrez le nom de la personne ou de l'animal
- **Obligatoire** : Sélectionnez un genre (Femme 👩, Homme 👨, Non renseigné ❓, Chat 🐱, Chien 🐶, etc.)
- **Optionnel** : Choisissez une relation familiale (Père 👨, Mère 👩, Fils 👦, Fille 👧, etc.)
- **Obligatoire** : Choisissez une couleur unique pour cet utilisateur avec le sélecteur de couleur (colorimètre). Vous avez accès à des millions de teintes !
- Cliquez sur "Créer l'utilisateur"

### Ajouter un rendez-vous
- Cliquez sur le jour où vous voulez ajouter un rendez-vous
- Sélectionnez l'utilisateur concerné
- **Obligatoire** : Choisissez une catégorie (Personnel, Familial, Médical, etc.)
- Entrez le titre du rendez-vous (obligatoire)
- Optionnel : ajoutez l'heure et une description
- Cliquez sur "Enregistrer"

**Les rendez-vous s'affichent avec l'icône et la couleur de leur catégorie**

### Modifier un rendez-vous
- Cliquez sur le rendez-vous existant dans le calendrier
- Modifiez les informations
- Cliquez sur "Enregistrer"

### Supprimer un rendez-vous
- Cliquez sur le rendez-vous à supprimer
- Cliquez sur le bouton "Supprimer"
- Confirmez la suppression

### Naviguer entre les mois
- Utilisez les boutons "◀ Mois précédent" et "Mois suivant ▶"
- Vous pouvez naviguer dans le passé et le futur indéfiniment

## 🎨 Couleurs Personnalisées

Chaque utilisateur peut choisir n'importe quelle couleur parmi les **millions de teintes disponibles** via un sélecteur de couleur intégré. Il n'y a plus de limites de palettes prédéfinies !

### Comment choisir une couleur

Lors de l'ajout d'un utilisateur, cliquez sur le sélecteur de couleur pour ouvrir le colorimètre et sélectionner la teinte que vous souhaitez. Vous pouvez :
- Entrer une valeur hexadécimale directe
- Cliquer dans le gradient pour choisir visuellement
- Utiliser les curseurs RGB ou HSL selon les navigateurs
- Sauvegardez votre choix en créant l'utilisateur

## 📅 Catégories de Rendez-vous

Chaque rendez-vous appartient à une catégorie avec une icône et une couleur distinctive :

- 👤 **Personnel** (Lila) - RDV personnels, coiffeur, etc.
- 👨‍👩‍👧‍👦 **Familial** (Lila pâle) - Événements familiaux, anniversaires
- 🏥 **Médical** (Rouge) - Rendez-vous médicaux, dentiste, vétérinaire
- 💼 **Professionnel** (Bleu) - Réunions de travail, entretiens
- 🎭 **Loisirs** (Orange) - Cinéma, théâtre, concerts
- 📋 **Administratif** (Vert) - Banques, assurances, administrations
- 🛍️ **Shopping** (Violet) - Courses, achats importants
- ⚽ **Sport** (Turquoise) - Entraînements, compétitions, médecins du sport
- ❓ **Autre** (Gris) - Toutes autres catégories

## 🛒 Liste de Courses Partagée

Chaque famille peut avoir sa propre liste de courses. Sélectionnez d'abord une famille active pour accéder à sa liste.

### Ajouter des articles
- Vérifiez qu'une famille est sélectionnée
- Cliquez sur le bouton "🛒 Liste de courses" dans la barre supérieure
- Tapez le nom de l'article dans le champ de saisie
- Appuyez sur Entrée ou cliquez sur "➕ Ajouter"

### Gérer les articles
- **Cocher/Décocher** : Cliquez sur la case à cocher pour marquer un article comme acheté
- **Supprimer un article** : Cliquez sur le "×" rouge à côté de l'article
- **Supprimer les articles cochés** : Cliquez sur "🗑️ Supprimer les articles cochés"

### Changer de famille
Pour accéder à la liste d'une autre famille :
- Utilisez le sélecteur de famille dans le panneau Familles
- La liste de courses se mettra automatiquement à jour

### Statistiques
La liste affiche automatiquement le nombre total d'articles et le nombre d'articles cochés pour la famille active.
